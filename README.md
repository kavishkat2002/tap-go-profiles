# SmartTap — QR & NFC Digital Profiles

SmartTap allows users to create and share digital profiles via QR code or NFC. Perfect for businesses, restaurants, and individuals.

## Tech Stack

- **Framework**: React (Vite)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Shadcn UI
- **Backend**: Supabase
- **Package Manager**: Bun

## Getting Started

1. Install dependencies:
   ```bash
   bun install
   ```

2. Run development server:
   ```bash
   bun dev
   ```

3. Build for production:
   ```bash
   bun run build
   ```
